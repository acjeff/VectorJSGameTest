**Sources**

Using this for everything vector related https://vectorjs.org

**Get Started**

1. npm install
2. Run build.sh
3. tsc -w
4. open index.html in browser

**Current Controls**

a/d or l/r arrows to move
space bar to jump
scroll to zoom in/out
