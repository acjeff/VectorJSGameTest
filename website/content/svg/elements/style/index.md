The style element defines CSS that is applied to elements through a combination of selectors.

{{< highlight html>}}
<style>
  /* select an individual element by id */
  #identifier {

  }

  /* select all elements with the class */
  .className {

  }

  /* select all elements with the tag name*/
  tagName {

  }
</style>
{{< /highlight >}}
